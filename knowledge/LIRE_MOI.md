# Base de Connaissances — Coach Parental

Tous les fichiers `.md` dans ce dossier sont automatiquement chargés et injectés
dans le contexte de l'IA à chaque réponse.

## Comment ajouter des connaissances :

1. Crée un fichier `.md` dans ce dossier (ex: `islam-parentage.md`, `tdah-strategies.md`)
2. Écris le contenu en français (ou arabe)
3. Pousse sur GitHub → Railway redéploie automatiquement

## Exemples de fichiers à créer :
- `islam-parentage.md` — conseils parentaux selon les valeurs islamiques
- `tdah-enfants.md` — stratégies spécifiques TDAH
- `maroc-education.md` — contexte culturel marocain
- `gestion-coleres.md` — protocole détaillé pour les crises

## Format recommandé :
```
# Titre du sujet

## Contexte
...

## Conseils pratiques
- ...

## À éviter
- ...
```
